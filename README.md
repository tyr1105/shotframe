## Features

- 16 Gradient Backgrounds — Aurora, Sunset, Ocean, Starry Night, Cyber, and more
- Solid Color and Pattern Backgrounds — Flexible background options
- Device Frames — Browser and phone mockups to showcase your screenshots
- Custom Border Radius — 0-48px adjustable
- Shadow Effects — Adjustable drop shadows for depth
- Watermark Text — Add custom branding or attribution
- HD Export — 1x, 2x, 3x, 4x resolution PNG
- Paste from Clipboard — Ctrl+V to paste screenshots directly
- 100% Private — All processing in-browser, nothing uploaded

## Quick Start

1. Visit [ShotFrame](https://tyr1105.github.io/shotframe/)
2. Upload or paste your screenshot
3. Pick a gradient background and adjust styles
4. Export as HD PNG — done!

No sign-up. No watermarks. No limits.

## Tech Stack

- Next.js 16 (Static Export)
- React 19
- Tailwind CSS 4
- Canvas API (client-side HD image generation)
