# 📸 ShotFrame - 截图美化工具

> 免费在线截图美化工具 | 渐变背景 · 设备框 · 阴影效果 · 一键导出高清PNG

🔗 **在线使用**: [https://tyr1105.github.io/shotframe/](https://tyr1105.github.io/shotframe/)

## ✨ 功能特点

- 🎨 **16种渐变背景** - 极光、日落、海洋、星空等精美渐变
- 🎯 **纯色/图案背景** - 灵活的背景选择
- 📱 **设备框** - 浏览器、手机等设备模拟
- 🌈 **自定义圆角** - 0-48px 可调节
- 💫 **阴影效果** - 可调节的投影效果
- 📝 **水印文字** - 添加自定义水印
- 📤 **高清导出** - 1x-4x 分辨率导出
- 🖼️ **多种上传方式** - 拖拽、粘贴、点击上传
- 🔒 **隐私安全** - 纯浏览器端处理，不上传任何数据

## 🚀 使用方法

1. 打开 [ShotFrame](https://tyr1105.github.io/shotframe/)
2. 上传截图（拖拽/粘贴/点击）
3. 选择背景、调整样式
4. 点击"导出 PNG"下载

## 🛠️ 技术栈

- Next.js 16 (Static Export)
- React 19
- Tailwind CSS 4
- 纯客户端处理，无需后端
